# conversion-optimization

Design and run A/B tests to improve content performance, engagement, and conversion metrics.

## When to Use

Use this skill when you need to:
- Plan A/B tests for landing pages, emails, or social content
- Define success metrics and sample sizes
- Analyze test results and calculate statistical significance
- Iterate on creative based on performance data

## How It Works

This lightweight framework guides you through:

1. **Test Design:** Define control vs. variant, hypothesis, and metrics
2. **Sample Size:** Calculate required sample based on baseline conversion rate and lift target
3. **Duration:** Estimate test duration given traffic volume
4. **Analysis:** Interpret results and determine if difference is significant
5. **Iteration:** Document learnings and plan next test

## Example Workflow

1. Choose what to optimize (email subject, CTA copy, image, timing)
2. Define baseline metric and target lift
3. Calculate sample size and test duration
4. Run test with equal traffic split
5. Analyze results and document learnings
