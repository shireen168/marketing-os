## The 6 Stages

Marketing OS operates across 6 stages with clear handoffs between tiers:

1. **Tier 1 (Strategic)** — Goals, audience, positioning
2. **Tier 1→2 (Gate)** — Executive approval checkpoint
3. **Tier 2 (Approval)** — Campaign structure, messaging framework
4. **Tier 2→3 (Gate)** — Quality gate before execution
5. **Tier 3 (Execution)** — Creative production, content calendar
6. **Measurement** — Metrics, iteration, optimization
