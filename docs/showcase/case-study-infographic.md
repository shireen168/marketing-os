# Case Study: Brand-Compliant Infographic in 30 Minutes

## The Problem

Branded infographics required either hiring a designer or spending **3+ hours in Figma** learning design constraints from scratch every time. This created friction:

- No consistency — font sizing, color application, spacing varied asset to asset
- Slow iteration — describe → mockup → revise → export took days
- High activation energy — I'd skip infographics for months rather than deal with the friction
- Manual QA — every asset needed human review to ensure brand compliance

The real cost wasn't just time — it was the inability to ship visual content on my own timeline.

## The Solution

A locked infographic system that separates **judgment** (what to illustrate, how to frame data) from **production** (fonts, colors, spacing, QA).

**The system:**
```
Describe concept → System structures layout → HTML → PNG export
```

**Locked constraints (automated):**
- Font: Space Grotesk with enforced weights and sizes for hierarchy
- Colors: Petrol #0099CC on dark background, consistent contrast ratios
- Spacing: Baseline grid with automated padding and alignment
- QA gate: Automatic compliance scoring; rejects anything below 80/100

**What I control (judgment):**
- Topic and key message — what the infographic communicates
- Layout pattern — which of 6 templates best fits the data
- Final review — eyeball the output before publishing

**What the system controls (production):**
- Typography, color application, spacing, hierarchy
- Compliance verification
- Export (PNG for web/social, PDF for print)

## The Result

**30 minutes per infographic.** From concept to export-ready asset. The system has produced 4+ infographics over the past quarter with 100% brand consistency.

| Metric | Manual Process | With System |
|--------|---|---|
| Time per asset | 2–3 hours | 30 minutes |
| Brand compliance | Manual QA (variable) | Automated 95/100 gate |
| Revision latency | 24–48 hours | 5 minutes |
| Monthly capacity | 2–3 assets | 4–8+ assets |

## Key Takeaway

This isn't about cutting costs or avoiding designers. It's about separating what requires human judgment from what requires consistency. Once the brand constraints are locked, I can iterate on messaging and framing without friction. That's how a one-person operation ships on-brand visual content at scale — not by automating creativity, but by automating the production guardrails so creativity is faster.

This scales the output (4+ assets/month instead of 2–3) without requiring a designer hire or sacrificing brand integrity.
