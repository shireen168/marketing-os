## Layer Descriptions

### 1. Strategic Discovery
Foundation research and brand definition. Answer "Who are we?" and "Where do we compete?"

### 2. Creative Planning
Content strategy and messaging. Answer "What will we say?" and "How often?"

### 3. Execution
Production and distribution. Answer "What format?" and "Who sees it?"

### 4. Optimization
Performance tracking and iteration. Answer "Did it work?" and "How do we improve?"

## Usage Example Workflow

1. Start with **Strategic Discovery** (brand-guidelines, market-research, audience-research)
2. Move to **Creative Planning** (campaign-plan, content-calendar-strategy, copy-gen)
3. Execute with **Execution** tools (video-gen, email-sms, social-scheduling, analytics)
4. Refine with **Optimization** (conversion-optimization, cost-roi-tracking)

For ongoing content:
- Use content-calendar-strategy to plan themes
- Use copy-gen + video-gen to create assets
- Use social-scheduling to publish
- Use analytics to track performance
- Use conversion-optimization to test improvements
